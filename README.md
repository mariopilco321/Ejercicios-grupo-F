Este proyecto contiene una colección de ejercicios en C++ y Java enfocados en la detección y corrección de errores comunes en programación. Los ejercicios trabajan conceptos como arreglos, ciclos for, búsqueda lineal, validación de índices, comparación de cadenas, contadores y control de flujo. Cada código fue corregido para evitar problemas de sintaxis, desbordamiento de arreglos y errores lógicos, mostrando buenas prácticas básicas de programación y depuración.
